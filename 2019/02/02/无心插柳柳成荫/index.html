无心插柳柳成荫

发表于 2019-02-02 | 分类于 随笔

文学、美术、音乐、电影、摄影，我似乎样样都沾了点儿，这一两年，也算是不知不觉中活成了半个文艺青年的模样吧？但真正的文艺青年，从来不当自己是什么文艺青年。

书没读几本，倒是喜欢到处写文章骂人，大声嚷嚷，活像个kibitzer，以前乱用的写作技巧，后来在语文课上才知道叫博喻。小学美术课次次拿A+，到现在除了画笑脸连简笔画都画不好，更别提什么构图光影，对着几张破纸瞎谈什么解构主义达达主义，正所谓：万物皆可后现代。几年下来歌是听了上万首了，从流行到实验，从民谣到金属，不可谓不杂，虽说各种各样的rock是最爱，见得多了也会道点门路来，可惜至今分不清什么黑金死金工业的东西，古典听得也少，最遗憾的是上手只会敲几个动次打次，按几个基本和弦，弹几个哆来咪嗦，半桶水都高估我了。对荧幕有谜之信仰，搞得文艺片没看几部，好不容易有好的剧情片看了想写点什么，提起笔来空流两行泪，长镜头？还是漫威让我最爽。摄影？不会。微单都没碰过，拿手机瞎拍，设备不够，构思来凑，构图随天气构，布光看心情布，长曝光这种东西是玩不起了，反正最后拿荒诞主义糊弄人就行。

罢了，全才和专才的问题从民国争到现在都没有定论，我又怎知？以上种种，无非形式，皆为载体，思想内核才最令我看重，这是我的“形式论”。所以我是从来没有说自己“喜欢看动漫”云云，我爱某些作品，也对某些作品不屑一顾，甚至是感到恶心，产生差异的，不是载体，应是内核。此为同。

但毕竟不同的载体间还是有差异，我大概是最喜欢电影吧。为什么？“还原”地看，当然是因为它所涵盖的信息量大，画面（即变化的图像，涵盖美术摄影文学云云）、声音（配音、配乐和环境音），仅两者就容纳下了整个世界，表现力不可谓不丰富。常言“文字不死”，是因为思想不死；也说“文字苍白无力”，便是因为载体太过单薄。此为异。

所以，非要给我贴上个“文艺青年”的标签，我不算是很抗拒的；但我之所以发挥着我的价值，不是因为我现在正在写着这么一篇烂文章，而是因为我的思想闪着金光。奈万物变更，思想永存。

大抵如此。

二月二日凌晨。腊月廿八，快过年咯。
